# 空間図形の断面ビューア

TypeScript + Vite + Three.js で作った、立体を切断平面で切って断面を観察する学習用Webアプリです。

## 起動方法

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開くとアプリが起動します。

## 操作方法

- 3D表示エリアをドラッグすると立体を回転できます。
- ホイールまたはピンチ操作で拡大・縮小できます。
- 右ドラッグで表示位置を移動できます。
- 「切断位置」スライダーで切断平面を前後に動かせます。
- 「X軸回転」「Y軸回転」「Z軸回転」スライダーで切断平面の向きを変えられます。
- 「表示する立体」から、立方体、直方体、円柱、円錐、球を選べます。
- 「内側に図形を入れる」をオンにすると、外側の立体の中へ別の立体を入れられます。
- 外側と内側の立体は同じ切断平面で切られ、3D表示と2D断面図の両方に表示されます。
- 立体は底面がグリッド面にそろうように配置されています。
- 立体上の白い点をクリックすると、切断面を作る点として選択できます。
- 3点を選ぶと仮の切断面を表示し、4点が同一平面上にある場合は、その4点を通る切断面で切断できます。
- 4点が同一平面上にない場合は、警告を表示します。
- 選択した点をもう一度クリックすると選択を外せます。「点の選択をクリア」でも選択をリセットできます。
- 「半透明表示」で立体の内側を見やすくできます。
- 「切断平面を表示」で半透明の板とPlaneHelperの表示を切り替えられます。
- 「スクリーンショット保存」で現在の3D表示をPNGとして保存できます。
- 「プレゼンテーションモード」で3D表示を全画面化し、断面図だけを重ねて表示できます。
- 「リセット」で切断位置と角度を初期状態に戻します。

## 実装メモ

- 実際のメッシュ分割やブーリアン演算は行っていません。
- Three.js の `clippingPlanes` と `renderer.localClippingEnabled = true` を使って、立体が切られているように表示しています。
- 切断位置の表示は巨大な平面板ではなく、立体と切断平面の交線から作った断面キャップで表示しています。
- 2D断面図は表示中のメッシュと切断平面の交線から計算しています。
- 円柱、円錐、球はメッシュ近似の輪郭を使って、円・楕円・多角形として表示します。
## 追加機能

- 内側に入れる図形は、外側の図形の大きさに合わせて自動的に内接するように配置されます。
- 「プレゼン用ウィンドウを開く」を押すと、通常の操作画面とは別に、見せる用のウィンドウを開きます。
- プレゼン用ウィンドウには操作パネルを表示せず、3D図形と断面図だけを表示します。
- 通常の操作画面で図形、内側図形、切断位置、切断角度を変えると、プレゼン用ウィンドウにも反映されます。
- 「消す側」から、切断平面のどちら側を非表示にするかを切り替えられます。
